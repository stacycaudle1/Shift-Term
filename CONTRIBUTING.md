# Contributing to Shift-Term

Thank you for your interest in contributing to Shift-Term!

## How to Contribute

1. Fork this repository
2. Clone your fork locally
3. Create a feature branch from `updates`
4. Make your changes
5. Commit with a clear message
6. Push to your fork
7. Open a Pull Request targeting the `updates` branch

## Branch Strategy

- `main` – stable, protected releases
- `updates` – active development

Pull requests should normally target `updates`.

## Code Style & Expectations

- Keep changes focused and readable
- Comment non-obvious logic
- Follow existing project structure

## Getting Help

If you’re unsure where to start, open an Issue or look for issues labeled `good first issue`.
