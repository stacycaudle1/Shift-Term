# Assets

## Icon Requirements

To complete the Windows build, you need to add an icon file:

### Required File:
- `icon.ico` - Windows icon file (256x256 pixels recommended)

### Creating the Icon:
1. Create a 256x256 pixel PNG image with your app icon design
2. Convert to .ico format using an online converter or tool
3. Place as `assets/icon.ico`

### Suggested Design:
- Terminal/screen motif 
- Retro/vintage computing theme
- High contrast colors for visibility
- Simple, recognizable at small sizes

### Online Converters:
- https://convertio.co/png-ico/
- https://icoconvert.com/
- https://www.favicon-generator.org/

Until you add the icon, the build will use Electron's default icon.
